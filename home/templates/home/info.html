Git Test Help
  • The purpose of this website is to test the possibility of version control to be used in general-purpose file storage/editing.
  • This website is very likely to break due to it being a prototype, please put any bugs or problems you find in the feedback form!
  • If you have any queries, such as how to use some git features (though I encourage you to play around with the website first), then contact me at gxb199@student.bham.ac.uk
  • Do not, I repeat, do NOT use this website to store anything long term! This website is merely a proof-of-concept. Expect any and all data to be lost at any time without warning.
  • Merge requests here are not real merge requests, they are merely provided to see what kind of merge conflicts would be created if you were to merge.
  • If there is a merge conflict, you have to resolve it. You can't do anything else in the meantime.
  • If you are not considering using this website again, please delete your account so that you can free up space on my webserver.
  • Do not worry about the "collaborators" option on the repository section. It doesn't do anything.
